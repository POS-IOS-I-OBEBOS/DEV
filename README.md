# Game Dev Simulator (Python)

Небольшой учебный симулятор управления инди-студией. В проекте есть базовые модели
(студия, сотрудники, проекты, рыночные тренды), недельный игровой цикл, текстовый
CLI и обновлённый визуальный интерфейс на pygame в стиле дашборда менеджера студии.

## Возможности
- Недельная симуляция через `GameSimulation` (зарплаты, прогресс проектов,
  события, релизы, тренды рынка).
- Текстовый интерфейс `GameCLI` для управления студией: найм/увольнение, создание
  проектов, назначение сотрудников, отчёты, сохранение/загрузка.
- Визуальный интерфейс `GamePygameUI` с карточками сотрудников и проектов,
  логом событий, модальными окнами (найм, новые проекты) и авто-симуляцией.
- JSON-сохранения состояния (студия, проекты, сотрудники, баланс) через
  `save_game` / `load_game`.
- Базовые юнит-тесты на встроенном `unittest`.

## Требования
- Python 3.10+
- Для pygame-UI нужен `pygame` (в CLI-режиме внешние зависимости не требуются):

```bash
pip install pygame
```

## Запуск
Перейдите в корень репозитория и выполните одну из команд:

- Текстовый интерфейс (по умолчанию):
  ```bash
  python -m game_dev_simulator.main
  ```
- Визуальный интерфейс на pygame (красивый дашборд):
  ```bash
  python -m game_dev_simulator.main --gui
  ```

### Управление
- CLI: используйте пункты меню, чтобы продвигать недели, управлять сотрудниками
  и проектами, сохранять/загружать игру (`save/game_save.json`).
- Pygame: кнопки в правой панели для действий, `Space/Enter` — следующая неделя,
  `Esc`/`Q` — выход, прокрутка списков колесом мыши, выбор сотрудника/проекта по
  клику.

### Визуальный интерфейс
- Верхняя панель — студия, год/неделя, деньги и репутация.
- Левая колонка — карточки сотрудников с зарплатами и усталостью (скролл).
- Центр — проекты с прогресс-барами и статусами.
- Правая колонка — действия (неделя, авто-симуляция, найм/увольнение,
  новые проекты, назначение на проект, сохранение/загрузка, выход).
- Нижняя панель — лог событий (релизы, найм, неделя и т.д.).

## Сохранения
- Сохранить: пункт «5) Сохранить игру» в `GameCLI`.
- Загрузить: пункт «6) Загрузить игру» (файл по умолчанию — `save/game_save.json`).

## Тесты
Запустите юнит-тесты командой:
```bash
python -m unittest discover -s game_dev_simulator/tests
```
